# VX Racer

Небольшая гоночная игра в разработке. Этот репозиторий сейчас содержит только описание и план улучшений, чтобы двигаться от прототипа к более полной версии.

## Что добавляем дальше
- **Карьерный режим:** этапы с целями и наградами, прогрессия игрока и улучшения машины.
- **Гонка на время:** локальные и онлайн-таблицы лидеров, фантом-противник и отдельный экран выбора трассы.
- **Гараж и кастомизация:** окраски, винилы, диски и другие косметические опции с сохранением пресетов.

Подробнее о реализации этих пунктов — в файле [`docs/FEATURES.md`](docs/FEATURES.md).
Если у вас уже есть исходники и сборка EXE, следуйте практическому гиду по внедрению: [`docs/IMPLEMENTATION_GUIDE.md`](docs/IMPLEMENTATION_GUIDE.md).

## Как помочь
1. Ознакомьтесь с планом в `docs/FEATURES.md`.
2. Выберите блок, который хотите реализовать, и опишите его в задачах/issue.
3. Делайте небольшие PR с отдельными фичами (например, сохранения, таблица лидеров, экран гаража).

Спасибо за интерес к проекту!
